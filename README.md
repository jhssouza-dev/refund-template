# Refund — Expense Reimbursement System

A minimal and clean web application for submitting and managing reimbursement requests.  
Built with **HTML**, **CSS**, and **JavaScript**, using the **German Euro currency format** (e.g., 1.234,56 €).

---

## Features
- Add expenses with name, category, and amount  
- Automatic German-format currency input (1.234,56 €)  
- Dynamic expense list  
- Remove items individually  
- Auto-updating total expenses and amount  
- Fully responsive interface  

---

## Technologies
- HTML5  
- CSS3  
- JavaScript (ES6+)  
- German locale currency formatting (`de-DE`, EUR)  

---

## How to Run
1. Clone the project:  
   `git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git`

2. Open the folder:  
   `cd YOUR_REPOSITORY`

3. Run the app:  
   - Open `index.html` in your browser, **or**  
   - Use **Live Server** (recommended)

---

## Project Structure
- `index.html` — Interface  
- `styles.css` — Layout and styling  
- `scripts.js` — Logic, dynamic list, currency formatting  
- `/img` — Icons and assets
